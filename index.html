Project Examples

Twitter Sentiment Analysis

This notebook calculates a sentiment score for any business name or relevant social topic based on recently published tweets. The topic is chosen by the user before related tweets are analyzed using Natural Language Processing. A ‘sentiment score’ between 1-5 is returned to the user along with examples of tweets used during the analysis. This project leverages data from the Twitter API.

Media Consuption Analysis

This project examines which historically important voter issues Americans are most focused on today and if those issues have changed when compared to recent election years. The analysis uses data from the New York Times API to visualize trends in publication frequencies across the chosen topics. Python’s Pandas and MatPlotLib libraries are used to analyze and visualize the data. Markdown and analysis are included in this notebook.

Python API Challenge

Google and OpenWeatherMap API’s are used to visualize weather conditions of 500+ cities across the world. Weather conditions are compared based on varying distance from the equator. Summary statistics and linear regressions are used to determine which climate factors are having the largest impact on current weather conditions. This notebook contains examples of the statistical analysis.

SQL Alchemy Challenge

In this project the SQLAlchemy library is used to analyze Hawaii’s weather data which is housed in a database. The ORM model is used to query the database using object-oriented programming (climate.ipynb). In addition, the Flask library is used to build an API (app.py) that returns Hawaii’s weather data based on the user’s custom parameters.
