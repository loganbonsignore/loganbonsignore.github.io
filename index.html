Project Examples

Twitter Sentiment Analysis

This full-stack web app leverages Twitter's data to give decision makers an easy way to gauge market sentiment towards their product or brand. Sentiment is determined using Machine Learning and Natural Language Processing. Upon user input of a business name and location, the latest tweets related to that business are retrieved from Twitter's API before the app calculates a sentiment score between 1-5. The app returns this sentiment score along with examples of tweets used during the analysis and other information about the business including hours, location, reviews, pictures, etc. Raw tweet data is collected from Twitter's API at the time of user interaction.

Media Consuption Analysis

This project examines which historically important voter issues Americans are most focused on today and if those issues have changed when compared to recent election years. The analysis uses data from the New York Times API to visualize trends in publication frequencies of content related to historically important voter issues. The impact of Covid-19 on coverage of the upcoming presidential election is also explored. Python’s Pandas and MatPlotLib libraries are used to analyze and visualize the data. Markdown and analysis are included in this notebook.

Foodies API

The project focuses on providing Colorado business data to interested parties through an API. A database of 7000 businesses in Colorado is constructed using web scraping and Yelp's API. A detailed breakdown of the ETL process can be found here. We built our own API to allow users to retrieve business specific data from our database based on custom queries. The user has the option to narrow their search based on business name, city, zip, etc. The API was built using Python and Flask. Selenium, BeautifulSoup and Chromedriver were used to scrape data from the web. All data is housed in a MongoDB database.

Weather API

Google and OpenWeatherMap API’s are used to visualize weather conditions of 500+ cities across the world. Weather conditions are compared based on varying distance from the equator. Summary statistics and linear regressions are used to determine which climate factors are having the largest impact on current weather conditions. This notebook contains examples of the statistical analysis.
