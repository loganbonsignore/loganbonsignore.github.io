Project Examples

Twitter Sentimate Analysis

This app calculates a consumer sentimate score for any business or popular topic based on positive or negative sentimate in recently published tweets. The topic is chosen by the user before related tweets are requested from Twitter's API and analyzed using Natural Language Processing to identify positive or negative sentimate. A sentimate score is calculated and returned to the user along with a few examples of tweets used during the analysis.

Media Consuption Analysis

This project examines how Covid-19 is changing America's media consumption habits. The analysis is focused on identifying how demand for publications on historically important voter issues has changed in today's environment compared to recent election years. Data from the New York Times API is used during the anlysis.

Python API Challenge

Pandas and MatPlotLib are used to analyze weather data pulled from the OpenWeatherMap API. Linear regression and r-squared values are used to determine which climate factors are having the largest impact on current weather conditions.

SQL Alchemy Challenge

SQL and Python are combined using the SQLAlchemy library. Weather station data from Hawaii is analyzed to find the ideal vacation timeframe. In addition, the Flask library is used to build an API that returns weather data based on the user's request.
